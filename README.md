# omi-sinhala

A modern, Vercel-ready web interpretation of the Sinhala card game Omi.

## Features

- Four-seat Omi table with one human player and three AI seats
- Trump choice after the opening four cards
- Legal play enforcement with follow-suit rules
- Trick resolution with trump handling
- Token scoring with defender bonus, kapothi bonus, and 4-4 carry bonus
- Match flow to 10 tokens

## Local run

```bash
npm run dev
```

Open `http://localhost:4180`.

## Tests

```bash
npm test
```
